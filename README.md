# Adonis Rabbit MQ

[Rabbit MQ](https://www.rabbitmq.com/) provider for AdonisJs framework version 4.1.

### Compatibility

*This package has been built and tested on Adonis 4.1 and maybe incompatible with Adonis 3 and earlier or Adonis 5 and later.
